SP‑API Quick Site
=================

Minimal site for Amazon SP‑API developer profile.

Customize:
- index.html, privacy.html, terms.html placeholders:
  brand = Your Store Ltd.
  site_url = https://yourdomain.com
  email = contact@yourdomain.com
  phone = +1-555-0100
  location = Ontario, Canada

Deploy (fast):
- Netlify: https://app.netlify.com/drop (drag & drop folder)
- GitHub Pages: push to repo → Settings → Pages → Deploy from a branch.
